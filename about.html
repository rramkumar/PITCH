About this Program

PITCH (Personal Intonation Trainer, Computer Heuristics), the UNC Bands Intonation Trainer, is designed to help musicians hear pitch discrepancies in real time, as one would need to do while playing in rehearsal or performance.

We are grateful to Robert Spradling, Jay Berger, and Carl Fischer Music for granting permission to use the arrangements from the textbook Error Detection: Exercises for the Instrumental Conductor.

All of the performances are acoustic recordings of UNC undergraduate musicians that have been auto-tuned in the studio to be in-tune, sharp, or flat. We used Pro Tools to record tracks, Antares Auto-Tune Live to adjust their pitch, Logic Pro X to mix and master, and Audacity to create the drones. We recommend using headphones for highest fidelity.

On the Home Page, after the user chooses a melody and orchestration the program generates an example with the selected instruments along with a high drone and low drone. The drones are always in-tune and should be used for reference. All tracks can be muted or unmuted independently.

Depending on the selected level one or more voices will be out of tune. There are four gradations of sharp and four gradations of flat in addition to “in-tune.” The user must make a choice-- flat, sharp, or in-tune-- for all instruments. The program defaults to “in-tune.”

Alternatively, the Sandbox Mode allows the user to experiment with combinations of in-tune and out-of-tune for each voice to practice hearing what each possibility sounds like.

The application uses the following five arrangements:

  • Eine kleine Nachtmusik (Movement I) - W.A. Mozart / arranged by Andrew Balent
  • Fantasia in G Minor - J.S. Bach / arranged by Robert Spradling
  • Jupiter, The Bringer of Jollity (from The Planets) - Gustav Holst / arranged by Andrew Balent
  • Chester (Movement III from New England Triptych) - William Schuman
  • Simple Gifts - American Folk Hymn / arranged by Andrew Balent
  • Stereo Panning Test - a version of Simple Gifts for testing purposes
